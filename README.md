# Portfólio de Paulo Estalise

Este é o portfólio pessoal de **Paulo Estalise**, desenvolvido com HTML, CSS e JavaScript. O projeto apresenta informações sobre o autor, suas habilidades, projetos desenvolvidos, área de contato e uma área administrativa para adicionar certificados.

---

## 🔹 Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox e Grid)
- JavaScript (Vanilla)
- Font Awesome (ícones)
- Responsividade adaptada para mobile e tablet
